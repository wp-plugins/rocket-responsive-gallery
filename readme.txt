WordPress gallery plugin is a responsive gallery with focus on ease of use and best experience for visitors with any device.

== Description ==

Make your galleries stand out in an eye-catching and unique new way, Responsive  gallery  align images in justified grid keeping their original aspect ratio. You will be able to insert the gallery in any page or post with simple shortcode. Create unlimited responsive media portfolios or galleries.

The WordPress gallery plugin comes by default with a beautiful, responsive, modern and configurable gallery skin with lightbox support. You can also extend the skin engine to use your own custom skins or get new skins with the [premium version](http://rocketplugins.com/wordpress-gallery-plugin/).

The plugin is designed for users and the UI is very intuitive and easy to use.There are no major functionality compromises made in the free version of the plugin if you do like the free version please purchase the [premium version](http://rocketplugins.com/wordpress-gallery-plugin/) of the plugin for extra goodies(listed below) and to support the future development of the plugin.

== Installation ==

1. Upload and extract the zip file downloaded to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click on the new menu item "WP Gallery" and create your first Gallery.
1. Use the gallery shortcode to insert the gallery into your posts or pages
